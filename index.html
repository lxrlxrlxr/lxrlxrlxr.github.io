vue-route

vue-router

[toc]

本地安装

main.js中

  1. 导入
  2. use一下
  3. 创建路由
  4. 挂载到vue实例上

配置规则

通过routes属性配置地址和路由管理的组件关系

main.js中

  1. 导入组件
  2. routes属性中进行配置关系
  3. path：设置地址
  4. component：设置组件
  5. 可以配置多个

路由出口

希望匹配到的组件显示在哪里，就在哪里设置一个router-view标签

声明式导航

通过router-link标签设置to属性为地址可以实现点击切换

编程式导航

组件中通过this.$router.push(地址)可以通过代码实现切换

路由传参

在地址后面写上?分隔：分隔两段数据

通过key = value & key2 = value的方式添加参数，value为获取数据的值

组件中通过this.$route.query访问对应的key即可获取数据

CSS布局

CSS布局

[toc]

css栅格布局 grid

开启grid布局

一个容器包含6个子元素
改成三列布局 grid-template-columns
  • 固定宽度

  • 浮动宽度 fr是grid专用单位，代表grid剩余空间

排列元素 grid-template-areas

对齐方式
  • 块轴垂直对齐：align-items
  • 行轴水平对齐：justy-items
  • 轨道垂直对齐：aligh-content
  • 轨道水平对齐：justify-content

Flex Box 一维布局方式

开启flex

默认子元素按行排列

Flexbox 布局是一维布局方式，要么按行排列，要么按列排列。

align-items 控制交叉轴，justify-content 控制主轴,始终不变。

改变水平方向对齐方式justify-content

改变交叉轴方向，垂直对齐方式align-items

  • 基线对齐 文字优先

flex-direction

设置空间占比

Flex-Basis

flex-basis是用来设置main size的。使用 flex-basis，可以同时管理行模式下的宽度和列模式下的高度。

此时width失效，但是可以用min-width限制

在列模式下，设置高度

缩放

flex-grow

flex 容器在有剩余空间的时候，子元素占据剩余空间的占比。适合一些浮动的布局。

flex-shrink

拆行

超过边界自动换行
