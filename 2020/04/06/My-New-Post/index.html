一、官网安装稳定版本node

https://nodejs.org/zh-cn/

检查是否安装好

二、注册github

选择create a project

建好以后做一个测试，点击create a new file

新建一个index.html文件

保存提交，点击commit new file

浏览器中打开lxrlxrlxr2.github.io

三、官网下载git

https://git-scm.com/downloads

检查是否安装好

四、githubSSH配置

  • 安装路径下打开git

    可以直接一直回车

  • 打开github / settings / SSH and GPG keys

    点击New SSH key

    title : 随便写

    key : C:\Users\Alice\.ssh\id_rsa.pub

    点击Add，即完成

五、本地hexo

官方文档 https://hexo.io/zh-cn/
